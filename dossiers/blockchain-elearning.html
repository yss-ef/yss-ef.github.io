System Objective

A Web 3.0 academic repository and e-learning system. Built with Solidity Smart Contracts on Ethereum and IPFS for decentralized storage, it ensures the immutability of academic records and peer-to-peer knowledge sharing.

Mission Post-Mortem

The Challenge

Managing the high gas costs of Ethereum for frequent academic updates and handling the asynchronous nature of blockchain transactions while providing a smooth UX in the React frontend.

The Solution

Optimized Smart Contract storage by only saving IPFS hashes (CIDs) on-chain rather than full document data. Implemented a robust event-listening system in React to provide real-time transaction feedback to users, ensuring they aren't left waiting during mining periods.
