AWS & Zabbix Infrastructure Monitoring

System Objective

A full-stack infrastructure supervision system deployed on AWS. Utilizing Zabbix, it monitors the health, performance, and security of distributed Linux and Windows servers, providing real-time alerting and visual dashboards.

Mission Post-Mortem

The Challenge

Securing monitoring traffic across public networks without compromising server performance. Configuring low-level discovery (LLD) to automatically detect and monitor dynamic AWS resources (EC2 instances, EBS volumes).

The Solution

Implemented PSK (Pre-Shared Key) encryption for all agent-to-server communications. Developed custom Zabbix templates with LLD scripts that use the AWS CLI to discover instance tags and auto-assign monitoring profiles, creating a self-healing monitoring mesh.
