System Objective

A unified CLI tool engineered to automate the local development lifecycle. devctl eliminates manual environment setup by orchestrating Spring Boot, Angular, and Docker-based databases through a single command interface.

Mission Post-Mortem

The Challenge

Managing parallel subprocesses for multiple languages (Java/TS) often leads to orphaned processes or port conflicts. Coordinating their startup sequence while providing clean console output was the primary hurdle.

The Solution

Implemented an asynchronous orchestration layer using Python's subprocess and threading modules. Added a robust environment scanner that validates prerequisites (JDK, Node, Docker) before execution, ensuring a 100% success rate on clean machines.
