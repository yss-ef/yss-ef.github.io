System Objective

A high-performance full-stack library platform for Mundiapolis University. Built with Next.js 15, it features a student portal for borrowing and an admin dashboard for inventory, fine management, and automated email reminders.

Mission Post-Mortem

The Challenge

Ensuring data consistency during peak hours when hundreds of students might try to borrow limited copies simultaneously. Managing the transition to React 19 and Next.js 15 features like Server Actions while maintaining type safety with Drizzle ORM.

The Solution

Implemented an optimistic concurrency control strategy using Drizzle transactions. Leveraged Upstash Redis for high-speed rate limiting and caching, ensuring the system remains responsive under heavy load. Automated fine calculation through a daily cron-job script.
